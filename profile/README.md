# Sparkfy Reviewer

O Sparkfy Reviewer é uma extensão de navegador especializada, projetada para aprimorar o processo de revisão de código no GitHub e GitLab utilizando Inteligência Artificial avançada.

Ao integrar modelos de linguagem de grande escala diretamente nos fluxos de trabalho de Pull Request e Merge Request, ele fornece feedback automatizado de alta qualidade baseado em boas práticas da indústria.
